# Звіт про використання AI

Для розробки мікросервісної архітектури було використано AI Gemini.

### Виконані завдання:
1. **Конфігурація Nginx:** Налаштування Rate Limiting та проксіювання запитів на порт 3000.
2. **Оркестрація:** Створення `docker-compose.yml` з поділом на внутрішню та зовнішню мережі.
3. **Оптимізація:** Налаштування Multi-stage Dockerfile для зменшення розміру образу до 187MB.
4. **DevOps:** Розробка `docker-compose.dev.yml` для автоматичного оновлення коду через volumes.
